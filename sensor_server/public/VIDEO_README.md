# Video Placeholder

Este directorio requiere un archivo de video para que el efecto "Video Reveal" funcione.

## Archivo Requerido
- **Nombre**: `videowall.mp4`
- **Ubicación**: `sensor_server/public/videowall.mp4`

## Instrucciones
1. Coloca tu archivo de video MP4 en esta carpeta.
2. Renómbralo a `videowall.mp4`.
3. Abre `http://localhost:3000/viewer.html` y haz clic en "CLICK TO START".

> **Nota**: El video no se incluye en el repositorio debido a limitaciones de tamaño de GitHub (max 100MB).
